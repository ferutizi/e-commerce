# e-Comerce

Una aplicacion de e-comerce, podes probar la demo haciendo click [ACA](https://github.com/)

## Configuracion inicial

Para correr el proyecto localmente:
```
$ npm install
$ npm start
```
La aplicacion localmente corre en el puerto 3000
